Actualizaciones

Desarrollo de páginas web: Participé en el desarrollo de diversas páginas web, desempeñándome principalmente como diseñadora. Mi rol incluyó la creación y personalización del diseño de las páginas, asegurando una experiencia visual atractiva y funcional. Los proyectos fueron desarrollados utilizando Visual Studio Code como entorno de desarrollo, y los lenguajes empleados fueron HTML5 y CSS .


Desarrollo de un videojuego: Formé parte del equipo encargado del desarrollo de un videojuego de sopa de letras, donde mi responsabilidad principal fue diseñar las interfaces del videojuego. Para este proyecto, también utilicé Visual Studio Code como herramienta de desarrollo, y trabajé con los lenguajes HTML5 , CSS y JavaScript .

Desarrollo de una casa inteligente: En este proyecto, desarrolló un rol multidisciplinario, participando tanto en la programación como en la conexión de circuitos y la construcción de la casa física. Utilizamos Arduino y componentes electrónicos como LEDs, servomotores, cables, resistencias y Raspberry Pi . Las tecnologías empleadas fueron:

  • Arduino : Programado en C++ .
  • Página web : Diseñada con HTML , CSS y JavaScript .
  • Comunicación con Raspberry Pi : Implementada utilizando Python y PHP para gestionar la interacción entre el hardware y el software.
  • Este proyecto me permitió adquirir experiencia en la integración de hardware y software, así como en la resolución de problemas técnicos relacionados con la domótica.
